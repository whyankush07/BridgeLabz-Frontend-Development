CSS Box Model Exploration

Understanding Content, Padding, Border, and Margin

What is the CSS Box Model?

The CSS box model is a fundamental concept that describes how elements are rendered on a web page. Every element is represented as a rectangular box, consisting of four distinct layers: content, padding, border, and margin.

Default Box Sizing (content-box)

Width and height apply only to the content area. Padding and border are added to the total size.

Margin (20px)
Border (5px solid)
Padding (30px)
Content

Width: 200px
Height: 150px

Total Size Calculation:

Width = 200px (content) + 60px (padding) + 10px (border) = 270px

Height = 150px (content) + 60px (padding) + 10px (border) = 220px

Margin is not included in the element's actual size

Border Box Sizing (border-box)

Width and height include content, padding, and border. This makes sizing more predictable and easier to manage.

Margin (20px)
Border (5px solid)
Padding (30px)
Content

Width: 200px
Height: 150px

Total Size Calculation:

Width = 200px (includes content + padding + border)

Height = 150px (includes content + padding + border)

Content area shrinks to accommodate padding and border
