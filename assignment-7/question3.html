Gradient Background Showcase

Exploring Linear and Radial Gradients with Parallax Effects

Linear Gradient Background

A linear gradient creates a smooth color transition along a straight line. It moves from one color to another in a specified direction.

This section features a diagonal linear gradient flowing from top-left to bottom-right, creating a dynamic visual effect.

  • Direction: 135 degrees (diagonal)
  • Colors: Purple → Pink → Orange
  • Effect: Smooth color transitions

Radial Gradient Background

A radial gradient radiates colors outward from a central point, creating a circular or elliptical pattern.

This section uses a radial gradient with a fixed background attachment, creating a parallax-like effect when you scroll through the page.

  • Shape: Circular gradient
  • Colors: Blue → Cyan → Teal
  • Effect: Parallax scrolling (background-attachment: fixed)

More About Gradients

CSS gradients are powerful tools for creating visually appealing designs without the need for image files, improving page load times and performance.

You can combine multiple colors, adjust gradient angles, and control color stops to achieve virtually any gradient effect imaginable.
