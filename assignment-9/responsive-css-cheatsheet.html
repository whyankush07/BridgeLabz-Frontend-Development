Responsive CSS Cheat Sheet

Filters

filter: grayscale(100%); /* try blur(), sepia(), hue-rotate() */

Media Queries

@media (max-width: 700px) { /* responsive rules */ }

Responsive Design

Use fluid layouts, relative units and CSS Grid / Flexbox to adapt.

Typography

h1 { font-size: clamp(1.5rem, 4vw, 3rem); }

Grid

grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

Flexbox

display:flex; align-items:center; justify-content:space-between;
